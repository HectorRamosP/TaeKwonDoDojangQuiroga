Class: ErrorBoundary

components/ErrorBoundary~ErrorBoundary(props)

Componente de clase que captura errores de JavaScript en el árbol de componentes hijo y muestra una interfaz de error amigable en lugar de que la aplicación se rompa. En modo desarrollo también muestra el stack trace del error.
